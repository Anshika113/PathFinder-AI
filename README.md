# 🚀 PathFinder AI  
### Intelligent Career Recommendation & Skill Analysis Platform

PathFinder AI is a full-stack AI-powered system designed to help students and job seekers make data-driven career decisions by analyzing skills, identifying gaps, and recommending optimal career paths.

---

## 🧠 Problem Statement

Most students:
- Don’t know which career suits them
- Lack clarity on required skills
- Follow random roadmaps from the internet

👉 PathFinder AI solves this by:
- Mapping skills → career roles
- Identifying missing skills
- Generating structured learning paths
